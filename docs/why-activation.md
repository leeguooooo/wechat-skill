# 为什么走人工审核制

简短答案：**控制分发、禁止滥用、保护账号生态**。

详细一点：

微信反自动化 + 账号风控一直在收紧。如果激活码完全开放下载，一周内会被刷成"群发营销 / 刷单 / 钓鱼 / 监控他人"的工具。一旦微信侧识别到我这套接入方式批量跑这类滥用流量，**所有用户**（包括你这种正常自动化场景的）的账号都会被一起波及。

所以每个激活码人工过一遍：

- 群发营销 / 刷单拉新 / 监控他人：**直接拒绝，不进入审核队列**
- 个人自动化（同步消息、给 AI agent 用、私人备份、群机器人）：**正常通过，1-24h**
- AI 星球成员：**备注星球身份 / 邀请人，走快速通道**
- 机构 / 商业场景：**目前不开放批量发放**，可以邮件单独商谈授权

申请时把"你是谁 + 想做什么"写清楚，通过会快很多。模糊的"想试试"或纯 emoji 通常会被打回再问。

---

## 我能不能跳过审核

不能。激活码是服务端权威 token：

- bridge 的每次 `/send` 都会校验 token 的 `expires_at` + tier
- daemon 的 hijack send 也会校验，**绕本地客户端没用**
- token 进 macOS Keychain（`wechat-skill-profile-api` service），不裸文件
- 撤销机制：管理员可以 server-side 把 token 标记 revoked，下次心跳即失效
